# haxsdk

A lightweight, high-performance C++ software development kit designed for building game modifications and internal tools.

- Windows x64
- C++ 20+

## Documentation

For detailed guides, API references and advanced usage, please visit an official documentation:

👉 **[Open](https://haxsdk.gitbook.io/en/)**

## Features

* **Unity engine support** Kit for interacting with Unity version 2019.4+ games on both **Mono** and **IL2CPP**.
* **Immediate-Mode GUI:** A lightweight graphical user interface module for building in-game menus (yet DirectX 11 only).
* **Zero-dependencies**: No external libraries used.
